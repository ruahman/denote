==setup android device==

Tools > Android > Android Device Manger
 * click on new
 * when you click on start is will ask you to enable hyper-v

==enable hyper-v==

search "windows features"
 * check hyper-v

the go to command prompt
#+begin_src bash
  systeminfo.exe
#+end_src
- check hyper-v requirements
- you may want to enable virtualization in bios


==setup your device==
got to Settings > About Phone > Build Number
 * click on this 7 times

then go to Settings > System > Developer Options
 * set USB Debugging or WiFi

change to Android Local Device

==setup emulator==
go to Tools > Android > Device Manager
